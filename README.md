# Organizer
I used to have folders that was always untidy, especially the `Desktop` and the `Downloads` folder. So I decided to make this small script to make my organization process faster and easier.

# How to use
1. Download the latest release.
2. Enter into the folder that you want to tidy.
3. Open CMD from that folder.
4. Run `Organizer.py` from that folder.

# Result
Before:
```
Downloads
├───Folder
│
├───Archive.zip
├───CppFile.cpp
├───Document.pdf
├───Image.png
├───Installer.exe
└───Video.mp4
```

After:
```
Downloads
└───Files
    ├───Archives

    ├───Documents
    │   └───Document.pdf
    ├───Folders

    ├───Other
    │   ├───CppFile.cpp
    │   └───Installer.exe
    └───Videos

```
